Coffee Shop Sample

Overview: This is a sample app which demonstrates basic Spring Data JPA repository usage with a Cloud Spanner database.

The interface below allows you to create/delete customers and coffee orders for each customer. All of the data interactions and persistence to Cloud Spanner is achieved via Spring Data JPA Repositories.

View all the saved data by opening a connection to the following JDBC URL:

jdbc:cloudspanner:/projects/test-project/instances/test-instance/databases/test-database?autoConfigEmulator=true

You can do this in IntelliJ by following the instructions here: IntelliJ - Other Databases. The Google Cloud Spanner driver is listed under Other Databases.

You can do this in DBeaver by following the instructions here: Steps to Connect to Emulator in DBeaver.
